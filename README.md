# Tetris_SDL2
<h2>A Clone of the Tetris Game in C++ using the SDL2 library</h2>
<br>
<image src = "https://github.com/ryland548/Tetris_SDL2/blob/main/screenshot.png">
<h2>Controls</h2>
<p>A - Move Tetris Piece <bold>Left</bold></p>
<p>D - Move Tetris Piece <bold>Right</bold></p>
<p>R - Rotate Tetris Piece </p>
<p>Space - Hard Drop</p>
<br>
<h2>Dependencies</h2>
What the project requires: <code><bold>SDL2</bold>, <bold>SDL2 ttf</bold>, <bold>SDL2 mixer</bold> and <bold>SDL2 image</bold></code>
<br>
<h2>Game Info</h2>
<p>The Score is calculated based on how high the line was cleared.</p>
