# Tetris_SDL2
<h2>A Clone of the Tetris Game in C++ using the SDL2 library</h2>
<br><br>
<image src = "https://github.com/ryland548/Tetris_SDL2/blob/main/screenshot.png">
<h2>Controls</h2>

<p>A - Move Tetris Piece <bold>Left</bold></p>
<p>D - Move Tetris Piece <bold>Right</bold></p>
<p>R - Rotate Tetris Piece </p>
<p>Space - Hard Drop</p>
<br><br>
<h2>Dependencies</h2>
<p><bold style="color: red;">SDL2</bold>, <bold style="color: red;">SDL2 ttf</bold>, <bold style="color: red;">SDL2 mixer</bold> and <bold style="color: red;">SDL2 image</bold></p>
